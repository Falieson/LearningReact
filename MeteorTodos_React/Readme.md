# GOAL
Meteor Boilerplate for Production Large Application


# RUNNING
## INSTALL
```
$ git clone git@github.com:Falieson/LearningReact.git
$ cd LearningReact
$ git co meteor_todos
$ cd MeteorTodos_React
$ meteor npm i -S
```
## TESTING
```
$ meteor test --driver-package practicalmeteor:mocha
```
## RUNNING
```
$ meteor
```
